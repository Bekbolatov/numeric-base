Release notes: StarPractice Alpha 0.2

Redesign of user interaction flow: now one can start activities immediately, without the need for two separate steps (install and run), reducing time between user load and first activity run.

Ability to star and add notes to individual questions, both currently viewed and last.

Filters on activity summary view: starred/with note/correct/wrong

Preparation for feature "tags" where inividual activities can be tagged and later looked up through a tag.

Incremental discovery of features: History link appears only when some activities were finished.

Tighter integration with server for activites list lookup - allowing one level of "confirmable" cache for faster presentation, later overwritten with fresher data from server if such exists.
